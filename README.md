# tfe

ClojureScript/OM implementation of 2048 [http://tfe.herokuapp.com](http://tfe.herokuapp.com)

## Running Locally

`
$ lein scss once
$ lein figwheel
`

## Running the tests (requires node to be installed)

`
$ lein test
`
